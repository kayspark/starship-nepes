# starship-nepes

Nepes corporate color palette for [Starship](https://starship.rs/) prompt — dark and light variants.

## Install

Copy the palette and module configs into your `~/.config/starship.toml`:

```bash
# Dark theme
cat nepes-dark.toml >> ~/.config/starship.toml

# Or light theme
cat nepes-light.toml >> ~/.config/starship.toml
```

Or selectively copy the `[palettes.nepes-dark]` section and set `palette = "nepes-dark"` at the top of your `starship.toml`.

## Colors

Part of the [nepes-palette](https://github.com/kayspark/nepes-palette) color system.
